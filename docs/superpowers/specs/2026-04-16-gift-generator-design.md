# Gift Generator — Design Spec
**Дата:** 2026-04-16  
**Проект:** Генератор ідей для подарунків  
**Ринок:** Україна  
**Статус:** Затверджено

---

## Концепція

Сайт-генератор ідей для подарунків для українського ринку. Користувач заповнює форму з параметрами отримувача — AI (Gemini) генерує 6 персоналізованих ідей у вигляді карток з посиланнями на пошук у Rozetka.

**Слоган:** *"Ідеальний подарунок за 30 секунд"*  
**Назва:** Дарунок / darunok.app (фінальна назва TBD)

---

## Ринок і конкуренти

- В Україні аналогів немає — лише каталоги магазинів (bodo.ua, gifty.in.ua)
- Англомовний ринок: GiftHub.ai, Gift Genius, Giftruly — всі через Amazon affiliate
- Нові слабкі місця конкурентів: тільки Amazon, нема локалізації, generic результати
- **Ніша в Україні повністю вільна**

## Форма — параметри

Всі параметри на одній сторінці. Вибір через **pill-shaped chips** (натиснув — підсвітилось), не dropdown.

| Параметр | Варіанти |
|---|---|
| Стать | Чоловік / Жінка / Без різниці |
| Вік | 0–12 / 13–17 / 18–35 / 35–60 / 60+ |
| Бюджет | до 300 грн / 300–1000 / 1000–3000 / 3000+ |
| Привід | День народження / Новий рік / Весілля / 8 березня / 14 лютого / Випускний / Просто так |
| Стосунки | Мама / Тато / Друг / Партнер / Колега / Дитина / Бабуся/дідусь |
| Хобі/інтереси | Спорт / Кулінарія / Ігри / Музика / Читання / Подорожі / Технології / Рукоділля / Кіно |
| Характер | Активний / Спокійний / Творчий / Практичний |
| Тип подарунку | Фізичний / Враження (похід кудись) / Цифровий |

## Flow

```
Користувач заповнює форму (все на одній сторінці)
       ↓
Натискає "Знайти подарунок"
       ↓
Next.js API route (/api/generate)
       ↓
Gemini API — генерує 6 ідей у JSON
       ↓
Відображаємо 6 карток з анімацією stagger
       ↓
Кожна картка: іконка + назва + опис + кнопка "Знайти на Rozetka"
       ↓
Кнопка формує URL: rozetka.com.ua/search/?text=<назва товару>
```

## Дизайн

**Стиль:** сучасний мінімалізм 2025 (референси: Vercel, Linear, Perplexity)

- **Темна тема** — фон `#0a0a0a`, текст `#ffffff`
- **Акцент** — фіолетово-рожевий градієнт (`#a855f7` → `#ec4899`)
- **Форма** — chip-кнопки з rounded corners, активний стан підсвічується акцентом
- **Картки результатів** — скляний ефект (glassmorphism), border `1px solid rgba(255,255,255,0.1)`
- **Анімації** — skeleton loader поки AI думає, stagger появи карток (Framer Motion)
- **Мобілка** — форма в один стовпець, картки в один стовпець

### Типографіка
- **Шрифт:** `Poiret One` (Google Fonts) — для заголовків і акцентних елементів
- Підключається через `next/font/google`

### CSS Variables (глобальні токени)
```css
--color-bg: #0a0a0a;
--color-surface: #111111;
--color-border: rgba(255, 255, 255, 0.08);
--color-text: #ffffff;
--color-text-muted: #888888;
--color-accent-start: #a855f7;
--color-accent-end: #ec4899;
--radius-md: 12px;
--radius-full: 999px;
```
